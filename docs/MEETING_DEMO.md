# Meeting demo

## Goal
Show the transition from old structure-generation scripts to the new reproducible software scaffold.

## What to show
1. Historical scripts used to obtain AFDB coverage and download CIFs
2. Domains as the current main local source
3. CIF validation result: 618 files, all ok
4. Foldseek real parsing on test5_vs_swissprot.m8
5. Current repo structure and state
6. How the paper helped frame the structural part of the project

## Key message
The project is no longer a scattered collection of scripts and folders.
It now has a local-first reproducible scaffold that already validates structures and parses real Foldseek results.

## Current limitation
Parser semantics were improved, but deeper biological interpretation still comes next.
